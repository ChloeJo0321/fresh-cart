# Frech Cart 🛒
A full-stack grocery shopping web application built with React, Node.js, Express, and MySQL.  
Designed to simulate a real-world online grocery shopping experience with cart management and automatic subtotal calculation.

## 🎯 Features
- Browse product list dynamically from backend API
- Add/remove items from shopping cart
- Update product quantity in real-time
- Automatic subtotal calculation
- RESTful API integration between frontend and backend

## 🛠 Tech Stack
- Frontend: React, CSS  
- Backend: Node.js, Express.js  
- Database: MySQL

## 🧠 Key Learnings
- Full-stack architecture (client-server separation)  
- REST API design and integration  
- State management in React  
- Database CRUD operations

## 🚀 How to Run
1. Clone repo 
```bash
git clone https://github.com/ChloeJo0321/my-grocery-app.git
cd my-grocery-app
```
2. Backend setup
```bash
cd backend
npm install
npm start
```

3. Frontend setup
```bash
cd frontend
npm install
npm run dev
```
